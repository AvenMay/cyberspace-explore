# PLC

## 一、 入门学习路径

### 1. 建立基础概念

在动手之前，你需要理解以下几个核心点：

- **硬件组成：** CPU 模块、输入模块（传感器、按钮）、输出模块（电磁阀、接触器、指示灯）。
- **工作原理：** 采用“循环扫描”机制（输入采样 -> 程序执行 -> 输出刷新）。
- **电气基础：** 了解什么是常开/常闭触点、继电器原理、低压配电知识（DC 24V 常用）。

### 2. 掌握编程语言

PLC 有五种标准语言（IEC 61131-3），初学者重点掌握前两种：

- **梯形图 (LD)：** 最像电路图，直观易懂，是工业界的通用语言。
- **结构化文本 (ST)：** 类似于 C 语言或 Python，适合处理复杂的数学运算。
- **功能块图 (FBD)：** 类似于逻辑门电路。

## 二、 硬件选择建议

不需要一上来就买几千块的工业级 PLC。根据预算和目标，有三种选择方案：

### 方案 A：高性价比入门（推荐）

- **品牌：** 台达 (Delta) 或 汇川 (Inovance)。
- **型号：** 台达 DVP 系列或汇川 H1U/H2U。
- **理由：** 价格便宜（二手几百元），中文资料极其丰富，编程软件免费，非常适合学习国产设备和基础指令。

### 方案 B：工业标准（进阶/就业）

- **品牌：** 西门子 (Siemens)。
- **型号：** S7-1200 或 二手 S7-200 SMART。
- **理由：** 市场占有率最高。学会了 S7-1200，你就掌握了 TIA Portal（博途）软件，这是目前工业自动化领域最高级的集成环境。

### 方案 C：极客/零成本方案

- **软件：** **OpenPLC**（开源项目）。
- **硬件：** 如果你有 Arduino 或树莓派，可以通过 OpenPLC 将它们变成兼容标准梯形图编程的“仿 PLC”。
- **仿真：** **Factory I/O**。这是一个 3D 虚拟工厂仿真软件，可以和西门子、三菱等软件联动，不需要买硬件也能看到传送带、机械臂在你的代码驱动下运行。

## 三、 推荐资源

### 1. 软件工具

- **西门子 TIA Portal：** 工业界的全家桶（体积巨大，对电脑配置有要求）。
- **三菱 GX Works3：** 日系 PLC 的代表。
- **PLC Fiddle (网页版)：** 一个简单的在线梯形图仿真器，适合前 10 分钟快速体验。

### 2. 在线课程与文档

- **Bilibili：** 搜索“西门子 S7-1200 入门”或“李金城 PLC”，有很多非常系统且免费的视频教程。
- **西门子 1847 工业学习平台：** 官方出品，内容极高质量。
- **PLC 论坛：** 工控网 (gongkong.com) 或 曹大侠工控论坛。

### 3. 书籍推荐

- 《PLC 编程理论、算法及技巧》
- 《深入浅出西门子 S7-1200 PLC》

## 四、 给初学者的 3 条建议

1. **不要只看书，一定要动手：** PLC 是实践学科。如果没有硬件，一定要装一个 **Factory I/O**，在虚拟工厂里调试程序比看 PPT 强一百倍。
2. **学会画电气图：** PLC 工程师不仅要写程序，还要画接线图（EPLAN 或 CAD）。看不懂图纸，程序写得再好也无法落地。
3. **从控制一盏灯开始：** * 第一步：按下按钮，灯亮；
   - 第二步：按下按钮，灯亮 5 秒后自动熄灭；
   - 第三步：两个按钮同时按下，灯才亮（逻辑与）。
   - 通过这些小目标，快速建立成就感。

# Emulator

GNS3 (Graphical Network Simulator-3): 支持模拟复杂的网络拓扑，兼容多种网络设备和操作系统，如Cisco IOS、Juniper JunOS等。

Cisco Packet Tracer: 由Cisco开发的模拟器，适用于学习和测试Cisco设备及其配置，用户界面友好，适合初学者。

EVE-NG (Emulated Virtual Environment Next Generation): 提供了一个图形化界面来创建和管理虚拟网络环境，支持多种网络设备和虚拟机的集成。

NS-3 (Network Simulator 3): 开源网络模拟器，适用于更复杂的网络模型和仿真需求，特别适合研究和开发。

Mininet: 主要用于SDN（软件定义网络）研究和开发，提供了一个轻量级的虚拟化环境来模拟网络拓扑。

# Application-Server

| **分类标签** | **框架名称** | **核心语言** | **设计哲学 / 特点** | **典型应用场景** |
| --- | --- | --- | --- | --- |
| **全栈 MVC (Batteries Included)** | **Spring Boot** | Java/Kotlin | 企业级标准，生态极度丰富，自动配置。 | 复杂企业后端、微服务 |
|     | **Django** | Python | 强调开发效率，“内置一切”，安全性高。 | 快速原型、内容管理、后台系统 |
|     | **Ruby on Rails** | Ruby | 约定优于配置（CoC），极速敏捷开发。 | 初创项目、高效率 Web 应用 |
|     | **Laravel** | PHP | 语法优美，拥有成熟的生态和工具链。 | 现代化 PHP 网站、中小型项目 |
| **轻量级/微框架 (Minimalist)** | **Flask** | Python | 极简主义，仅提供核心，插件灵活。 | 小型 API、胶水层、简单服务 |
|     | **Express.js** | Node.js | Node.js 事实上的标准，高度灵活、非阻塞。 | I/O 密集型应用、单页应用 |
|     | **Gin** | Go  | 极高性能，API 设计类似 Martini 但更快。 | 高并发微服务、高性能 API |
| **异步/响应式 (Reactive/Async)** | **FastAPI** | Python | 基于类型提示，原生支持异步，自动生成文档。 | 现代异步 API、机器学习模型部署 |
|     | **NestJS** | TypeScript | 借鉴 Angular 架构，强类型，适合大型 Node 项目。 | 健壮的后端架构、大规模 Node 服务 |
|     | **Vert.x** | Java/Polyglot | 全异步、多语言支持，极高的吞吐量。 | 实时通信、高吞吐分布式系统 |
| **高性能/云原生 (Cloud Native)** | **ASP.NET Core** | C#  | 跨平台，性能顶尖，与 Azure 云生态高度集成。 | 跨平台企业应用、高性能后端 |
|     | **Quarkus** | Java | 专为容器化设计，启动极快，内存占用低。 | 云原生、Serverless、K8s 环境 |

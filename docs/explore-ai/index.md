# Explore-AI

这个堆栈分为：**底层引擎（DL）、传统算法库（ML）、功能模块（CV/NLP）、超级应用接口（LLM）以及自动化生产线（AIE）**。

------

### 1. 核心底座：深度学习框架 (Deep Learning)

**理解视角：** 就像你开发桌面软件需要 .NET 或 Qt，开发网页需要 React/Vue，开发 AI 必须有底层引擎。

- **代表工具：**
  - **PyTorch：** 2026 年的绝对主流。代码写起来就像 Python 脚本，调试极快。
  - **TensorFlow / Keras：** 适合工业级的流水线部署。
  - **ONNX：** “跨平台协议”。如果你用 PyTorch 训练了一个模型，想在 Android 或浏览器里跑，通常先转成 ONNX 格式。
- **它负责什么：** 负责管理显存（GPU 内存）、张量计算（矩阵乘法）和自动求导（模型自我纠错）。

### 2. 传统逻辑：机器学习库 (Machine Learning)

**理解视角：** 处理的是“结构化数据”（类似 Excel 表格）。它不依赖庞大的神经网络，而是靠统计逻辑。

- **代表工具：**
  - **Scikit-learn：** “全能工具箱”。做分类、聚类、回归的万金油。
  - **XGBoost / LightGBM：** 金融行业风控、预测股市、电商推荐的最爱。
- **它负责什么：** 处理那些不需要“理解图片”或“理解语言”，只需要在数据中找规律的任务。

### 3. 垂直功能：CV 与 NLP 工具包

**理解视角：** 它们是针对特定媒体格式（图片、视频、文本）的“高级插件”。

| **领域**       | **现成工具/框架**              | **工程师拿它做什么**                                         |
| -------------- | ------------------------------ | ------------------------------------------------------------ |
| **CV (视觉)**  | **OpenCV** / **MediaPipe**     | 识别人脸、抠图、检测工业流水线上的次品。                     |
| **CV (高级)**  | **YOLOv10+** / **Ultralytics** | 极其迅速地识别视频里的车辆、行人、违禁品。                   |
| **NLP (文本)** | **SpaCy** / **NLTK**           | 自动提取合同里的日期、姓名，或者判断一段话是骂人的还是夸人的。 |

### 4. 超级大脑：LLM 时代的“新基建”

**理解视角：** 2026 年，如果你不想训练模型，只想**用**模型，你只需要知道这些“适配器”。

- **模型仓库：Hugging Face**。AI 界的 GitHub，你要什么模型（Llama, DeepSeek, Qwen）直接 `git clone` 过来跑。
- **运行环境：Ollama**。在本地 Linux/Mac 上跑大模型最简单的工具，一行命令启动。
- **推理加速：vLLM**。如果你要接上万个用户，它能帮你压榨显卡的每一份性能。

### 5. 工程纽带：AI Engineering (AIE)

**理解视角：** 这是把你熟悉的 **Docker、Airflow、Linux** 技能与 AI 结合的地方。

- **数据流编排：Apache Airflow / Prefect**。
  - *你的老朋友：* 定时抓取数据 -> 调用 CV 提取信息 -> 存入数据库 -> 触发通知。
- **记忆库（向量数据库）：Milvus / Pinecone**。
  - *新角色：* 专门存储文本的“向量”。当你问 AI 问题时，系统先在这里搜相关文档，再喂给 AI。
- **应用框架：LangChain / LlamaIndex**。
  - *粘合剂：* 它可以让 AI 自动查数据库、发邮件、写代码。你负责编排这些逻辑。

------

### 重新梳理：如果你要实现一个功能，怎么选工具？

| **你的需求**                             | **属于哪个领域** | **推荐使用的工具链**                                         |
| ---------------------------------------- | ---------------- | ------------------------------------------------------------ |
| **根据用户过往购买记录预测明天的销量**   | **ML**           | Python + Pandas + **Scikit-learn**                           |
| **识别监控录像里谁没戴安全帽**           | **CV**           | **YOLO** + **OpenCV**                                        |
| **建立一个公司内部的 Wiki 问答机器人**   | **LLM/NLP**      | **Ollama** (模型) + **Milvus** (向量库) + **LangChain** (逻辑) |
| **每天定时从网页抓取图片并自动生成描述** | **AIE**          | **Airflow** (调度) + **PyTorch/ViT** (模型)                  |

------

### 总结

你现在的理解可以停留在这一层：

1. **ML/DL** 是“底层开发语言和编译器”。
2. **CV/NLP** 是“多媒体处理库”。
3. **LLM** 是“高性能通用的云端/本地服务”。
4. **AIE** 是“如何把这些东西装进 Docker，用 Airflow 串起来并监控它”。

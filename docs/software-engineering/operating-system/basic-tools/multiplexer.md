## Windows Terminal Split

- 垂直分割 Alt + Shift + `+` 
- 水平分割 Alt + Shift + `-`
- 调整大小 Alt + Shift + `上下左右`
- 切换焦点 Alt + `上下左右`
- 关闭当前分块 Ctrl + Shift + W

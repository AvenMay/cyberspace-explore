## Introduction

### 操作系统学习的三重境界

| **维度** | **简称** | **核心关注点** | **核心工具/概念** |
| --- | --- | --- | --- |
| **应用层** | **Sysadmin** | **配置与策略**：如何让 OS 稳定、高效地运行起来。 | Shell (Bash/Zsh), Systemd, SSH, 权限管理 (RBAC), 包管理。 |
| **接口层** | **Sysprog** | **抽象与约定**：如何利用 OS 提供的能力构建底层软件。 | **POSIX API**, System Calls (fork, mmap, socket), 并发原语 (Pthreads)。 |
| **内核层** | **Internals** | **机制与实现**：内部的齿轮是如何咬合的。 | 调度算法 (CFS), 虚拟内存管理 (VMA), 中断处理, 文件系统实现。 |

- Sysadmin 主要是CLI/GUI得用户接口操作
- Sysprog 主要是对Kernel提供得API进行编程
- Internals 探索Kernel得原理以及API是如何实现的

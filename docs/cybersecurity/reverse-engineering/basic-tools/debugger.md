# Debugger

### 1. Windows 平台（主流必会）

Windows 是逆向工程的主战场（尤其是恶意软件分析和游戏逆向），以下两款是绝对的主力：

- **x64dbg (x32dbg)**
  
    - **定位：** **用户态（User Mode）首选**。它是 OllyDbg 的现代精神继承者。
      
    - **特点：** 开源、界面友好、插件生态丰富（如 ScyllaHide 用于过反调试）。它的操作逻辑非常符合逆向直觉，是目前 Windows 破解和各种 CrackMe 的标准工具。
      
    - **适用场景：** 应用程序逆向、恶意软件分析、Unpacking（脱壳）。
    
- **WinDbg (WinDbg Preview)**
  
    - **定位：** **内核态（Kernel Mode）权威** & 微软官方工具。
      
    - **特点：**
      
      - **内核调试：** 调试驱动程序、Windows 内核机制（你感兴趣的 OS Internals）的唯一真神。
        
      - **TTD (Time Travel Debugging)：** 杀手级功能。允许你记录程序的执行流并“倒带”回放，不需要复现 Bug，直接回退看内存状态。
        
      - **脚本能力：** 强大的 JavaScript 和传统的 MASM 命令支持。
        
    - **适用场景：** 驱动逆向、蓝屏分析、复杂的系统级漏洞利用、Rootkit 分析。
    

### 2. Linux / Unix 平台

- **GDB + 增强插件 (GEF / Pwndbg / Peda)**
  
    - **定位：** Linux 标准调试器。
      
    - **注意：** 原生 GDB 对逆向非常不友好。在 RE 和 CTF Pwn 领域，**必须**配合插件使用。
      
      - **GEF (GDB Enhanced Features)：** 现代、功能全面，支持多种架构（ARM, MIPS, PowerPC 等），对堆分析支持很好。
        
      - **Pwndbg：** 专为 Exploit 开发设计，不仅显示寄存器和堆栈，还能预测指令流，非常适合漏洞挖掘。
        
    - **适用场景：** Linux ELF 逆向、IoT 固件分析（配合 QEMU）、漏洞利用开发。
    
- **Radare2 (r2) / Rizin**
  
    - **定位：** 极客向、全能型 CLI 框架。
      
    - **特点：** "瑞士军刀"。它既是反汇编器也是调试器，支持极其广泛的架构。完全命令行操作，具有陡峭的学习曲线，但脚本化能力极强。
      
    - **适用场景：** 快速分析、Headless 环境、自动化脚本分析。
    

### 3. macOS / iOS 平台

- **LLDB**
  
    - **定位：** Apple 生态的标准调试器（Xcode 内置）。
      
    - **特点：** 替代了 GDB 在 macOS 上的地位。类似于 GDB，通常也需要配合插件（如 **Voltron** 或 **gef-lldb**）来改善逆向体验。
      
    - **适用场景：** macOS/iOS 应用调试。
    

### 4. 静态与动态结合（内置调试器）

这类工具主打“所见即所得”，让你在静态分析的代码上直接下断点，非常有助于理解复杂的控制流。

- **IDA Pro (Local/Remote Debugger)**
  
    - **地位：** 工业标准。
      
    - **特点：** 它的强大在于**反编译代码（F5）与动态调试的联动**。你可以直接在 C 伪代码上单步调试，变量值实时更新，极大地降低了理解汇编的门槛。
      
    - **适用场景：** 复杂的大型软件逆向、混淆代码分析。
    
- **Ghidra**
  
    - **地位：** NSA 开源的强力竞争者。
      
    - **特点：** 虽然早期调试功能较弱，但最新版本已经集成了调试器，支持通过 GDB/DbgModel 桥接调试，正在迅速追赶 IDA。
    

### 5. 动态插桩与特殊用途（非传统调试器，但 RE 必用）

- **Frida**
  
    - **本质：** 动态插桩工具（Dynamic Instrumentation）。
      
    - **RE 用途：** 它不给你看汇编指令的单步执行，但允许你用 JavaScript **注入**进程，Hook 任意函数，实时修改参数和返回值。
      
    - **适用场景：** 移动端安全（Android/iOS）、绕过 SSL Pinning、抓取加密函数的明文 Key、快速验证猜想。
    
- **Cheat Engine (CE)**
  
    - **本质：** 内存扫描与修改工具。
      
    - **RE 用途：** 虽然主要用于游戏修改，但其强大的内存扫描、指针扫描和内置的调试器功能，使其成为分析复杂数据结构和定位关键数据的神器。

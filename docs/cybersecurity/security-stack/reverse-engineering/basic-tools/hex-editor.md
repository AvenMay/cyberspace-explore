# Hex-Editor

### 1. 工业级与逆向分析首选 (Feature-Rich & Analysis)

这类编辑器不仅仅是看 Hex，更重要的是能够“解析”数据结构。

- **010 Editor** (Windows, Mac, Linux)
  
    - **特点：** 被称为 Hex 编辑器中的“瑞士军刀”。它最核心的功能是 **Binary Templates**（二进制模板）。你可以写简单的脚本（类 C 语法）来自动解析文件格式（如解析一个 `.exe` 头或 `.png` 块），直接把 Hex 映射成结构体字段查看。
      
    - **适用场景：** 逆向分析、分析未知文件格式、网络协议包分析。
      
    - **评价：** 只有它能让你把 Hex 当作“代码结构”来看，而不是枯燥的数字。
    
- **ImHex** (Windows, Mac, Linux)
  
    - **特点：** 现代化的开源编辑器，界面非常漂亮（GPU 加速），专为逆向工程师设计。它拥有类似 010 Editor 的模式语言（Pattern Language）来解析文件，支持数据预处理、反汇编预览等。
      
    - **适用场景：** 需要现代化 UI、夜间模式，且不想付费购买 010 Editor 的场景。
      
    - **评价：** 近年来最受开发者欢迎的开源新秀，更新非常快。
    

### 2. 轻量级与快速修改 (Lightweight & Editing)

这类工具主打打开速度快，能够处理超大文件（GB/TB 级别），适合快速修改或提取数据。

- **HxD** (Windows)
  
    - **特点：** 经典的免费工具，启动极快，界面朴素但功能扎实。支持直接打开内存（RAM）进行查看和编辑，也可以处理任意大小的文件。
        
    - **适用场景：** 快速修改某个字节、文件比较、磁盘镜像查看。
        
    - **评价：** Windows 平台最稳健的免费选择，“装机必备”。
    
- **Hex Fiend** (macOS)
  
    - **特点：** macOS 上最优秀的开源 Hex 编辑器。极其流畅，能够瞬间打开巨大的文件，支持二进制 Diff（比较）。
        
    - **适用场景：** Mac 用户做简单的 Patch 或查看二进制数据。
    
- **VS Code (Hex Editor Extension)** (跨平台)
  
    - **特点：** 微软官方出的插件。
        
    - **适用场景：** 偶尔需要看一眼 Hex，不想专门装个软件时非常方便。
    

### 3. 命令行与终端工具 (CLI / Terminal)

既然你对 Linux 和终端环境感兴趣，这些工具在服务器端或 SSH 环境下非常重要。

- **xxd/strings** (Linux/Mac)
  
    - **特点：** 系统自带（通常随 Vim 安装）。最常用的是将二进制转为 Hex 文本（dump），或者将 Hex 文本还原为二进制（patch）。
        
    - **命令：** `xxd file.bin` (查看), `xxd -r dump.txt > file.bin` (还原)。
    
- **hexedit** / **hexed** (Linux)
  
    
- **Radare2 (r2)** (跨平台)
  
    - **特点：** 虽然它是一个完整的逆向工程框架，但很多人直接用它做 Hex 编辑器。它的 `px` 系列命令非常强大。
        
    - **适用场景：** 极客、CTF 选手、不仅要看 Hex 还要看汇编指令时。
    

### 4. 电子取证专用 (Forensics)

- **WinHex** (Windows)
  
    - **特点：** X-Ways Forensics 的前身。它不仅是编辑器，更是磁盘取证工具，能直接读取物理磁盘、恢复删除文件、编辑内存。
        
    - **适用场景：** 数据恢复、系统取证。
